# Source Notes

**UNCLASSIFIED//OPEN SOURCE**

Source Notes preserves short provenance records for sources that need context
before they are reused. Each note records date, URL, source class, and why the
source matters.

See the [Collections hub](../README.md) and [Standards](../../standards/README.md) for overview and rules.

Source notes capture special handling instructions, access problems, attribution rules, and reconciliation notes so that later products reuse the source family correctly and consistently.
