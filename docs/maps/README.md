# Maps

**UNCLASSIFIED//OPEN SOURCE**

**Map Resource ID:** WI-MAPS-RESOURCE-DIRECTORY-2026-0001

**Prepared UTC:** 2026-06-14T20:37:05Z

**Information cutoff UTC:** 2026-06-14T20:37:05Z

**Purpose:** Provide a single navigation home for public map resources used
by WARLOCK-INDEX.

**Boundary:** This directory supports strategic orientation and source
provenance only. It does not provide targeting maps, sensitive facility
overlays, infrastructure vulnerability maps, live movement layers,
route-selection tools, collection geometry, patrol-pattern analysis,
interdiction guidance, operational planning, surveillance products,
investigative direction, or legal advice.

## Bottom Line

Maps belong in their own navigation lane. WARLOCK-INDEX should link to
public map sources from official or high-reliability publishers while keeping
map evidence separate from analytic conclusions. A map can orient a reader,
preserve publisher language, record command-area framing, or route a later
source packet. It does not independently prove sovereignty, control, threat
intent, military access, basing availability, vulnerability, or operational
feasibility.

This directory is the map-resource hub. It favors public map sources and map
portals over generated illustrations. When a map touches contested geography
or current conflict, products must preserve source language and state the
map's publisher, access date, product date where available, and intended use.

## Extraction Rules

1. Prefer public maps from official or high-reliability publishers.
2. Record publisher, URL, accessed UTC, publication or release date where
   available, product title, map family, and source purpose.
3. Label map use as static orientation, command-area framing, geographic-name
   support, nautical-chart source reference, topographic reference,
   historical/archival source, legal-source companion, or issuer perspective.
4. Preserve disputed names, boundaries, statuses, and source terms rather than
   silently normalizing them.
5. Pair maps with non-map evidence before making threat, coercion, access,
   basing, legal, control, or capability claims.
6. Keep generated images and decorative graphics out of the map-resource lane
   unless they are clearly labeled as illustrations and not source evidence.

## Prohibited Uses

- Targeting maps, target folders, or target-selection aids.
- Sensitive facility overlays, infrastructure vulnerability labels, access
  points, or base/port/airfield exploitation products.
- Route selection, interdiction points, patrol-pattern inference, evasion
  guidance, blockade mechanics, or fishing-enforcement guidance.
- Live vessel, aircraft, convoy, logistics, port, or troop movement tracking.
- Collection geometry, sensor coverage, watch zones, or surveillance tasking.
- Legal advice about maritime claims, treaty obligations, arbitration effect,
  or enforcement rights.
