Design Notes

The map remains the visual anchor while the marker layer stays light enough to maintain as static site content.

  • 151 KB WebP reference map; no tile server, canvas engine, or third-party map library.
  • Marker maintenance is isolated to one small JSON file.
  • Delegated click handlers for markers, filters, and list items.
  • No animation loops; hover is CSS-only and mobile hides tooltips.
  • `contain` and `content-visibility` limit layout and paint cost.
